# 🧠 Deep Learning Fundamentals

A hands-on repository documenting my deep learning journey with PyTorch. It includes practical notebooks, exercises, and examples covering tensor manipulation, autograd, GPU computing, and the foundational concepts required for modern AI and machine learning.

---

## 📖 About This Repository

This repository serves as a record of my progress as I learn the core concepts of deep learning using PyTorch. Rather than simply following tutorials, I focus on understanding the underlying principles through hands-on coding, experimentation, and problem-solving.

Each notebook builds upon the previous one, allowing me to develop a solid foundation before moving on to more advanced topics such as computer vision, natural language processing, and large language models.

---

## 🚀 What I've Learned

Throughout this repository, I have gained practical experience with:

- PyTorch tensors and tensor operations
- Tensor indexing, slicing, reshaping, and broadcasting
- Matrix multiplication and linear algebra fundamentals
- Tensor datatypes and device management (CPU & GPU)
- Automatic differentiation with Autograd
- Reproducibility using random seeds
- Working with CUDA for GPU acceleration
- Debugging common PyTorch errors
- Writing clean, readable, and reproducible PyTorch code
